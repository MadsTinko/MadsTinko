# Hi, I'm Mads Tinko Madsen 👋

**Frontend lead & design-focused engineer.** I build the systems other developers build *inside* — architecture, design systems, and the visual craft — with enough backend to lead across the whole stack.

- 🧭 ~10 years in frontend — building for the web since 2016
- 🎓 Apprenticeship-trained: web-integrater + Dateatechnician (with speciality in programming)
- 🎨 I specialise in frontend & design systems. Full-stack capable — but the visual work is where I do my best thinking.
- 🇩🇰 Based in Denmark

## What I focus on

- **Frontend architecture & design systems** — Angular, TypeScript, reusable component libraries, theming/design tokens
- **Design-minded engineering** — I care how an interface looks and *feels*, not just that it works; layout, responsiveness, and visual polish are where I'm strongest
- **Developer experience** — conventions, tooling, and lint rules that keep a team's codebase honest as it grows
